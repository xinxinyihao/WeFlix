## 🌟 核心特色

### 🎬 媒体库管理系统

WeFlix 采用先进的媒体库管理系统，为您的影视资源提供一站式解决方案：

- **智能媒体识别**：自动识别电影、电视剧、纪录片等不同类型的媒体文件
- **自动元数据获取**：智能刮削功能从网络获取片名、导演、演员、简介、评分等详细信息
- **精美海报墙**：展示高清海报和背景图，让您的媒体库焕然一新
- **智能分类系统**：按类型、年份、评分、国家/地区等多维度智能分类
- **批量处理工具**：支持批量扫描、刮削和整理，高效管理大量媒体文件

### 🚀 多源资源管理

WeFlix 突破传统媒体管理软件的局限，支持多种资源来源：

- **本地存储支持**：轻松管理本地硬盘、外接存储设备中的媒体文件
- **网络存储集成**：无缝支持 SMB/CIFS 网络共享，连接 NAS 和家庭服务器
- **资源源管理器**：直观的界面添加、编辑、删除和管理不同的资源来源
- **定时扫描**：自动检测新增内容，保持媒体库实时更新
- **状态监控**：实时查看各资源源的连接状态和扫描进度

## 💻 系统要求

WeFlix 支持多种平台，确保您随时随地都能访问您的媒体库：

- **Windows 系统**：Windows 10/11 64位
- **macOS 系统**：macOS 10.15 或更高版本
- **Linux 系统**：主流 Linux 发行版
- **内存要求**：最低 4GB RAM，推荐 8GB 以上
- **存储空间**：应用本身占用空间小，媒体库大小取决于您的收藏

## 📋 使用技巧

### 最佳实践

- **规范的文件命名**：使用"电影名 (年份)"格式命名电影，"剧集名/Season XX/剧集名 SXXEXX"格式命名剧集
- **定期扫描更新**：设置定时扫描，确保媒体库始终保持最新
- **备份元数据**：定期备份元数据数据库，保护您的媒体信息

### 进阶功能

- **自定义刮削规则**：针对特殊命名的媒体文件创建自定义刮削规则
